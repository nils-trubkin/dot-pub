# dot-pub

publicly available dotfiles for my personal use

## Requirements
- [GNU Stow](https://www.gnu.org/software/stow/)

## Installation

```bash
git clone git@github.com:nils-trubkin/dot-pub.git ~/.dot-pub
cd ~/.dot-pub
stow .
```

## Uninstallation

```bash
cd ~/.dot-pub
stow -D .
```
